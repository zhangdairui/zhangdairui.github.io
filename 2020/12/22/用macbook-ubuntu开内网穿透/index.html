用macbook+ubuntu开内网穿透

1.将 这个文件
放到ubuntu系统的电脑中。

2.运行
sudo tar xzvf frp_0.33.0_linux_amd64.tar.gz
解压

3.进入解压好的文件夹，修改frpc.ini配置文件，例如：


[common]
server_addr = 域名
server_port = 9000 //设定一个端口号
token = GCP111111 //令牌

[manager] //网站标识
type = tcp
local_ip = localhost
local_port = 7000 //设定端口号
remote_port = 7000

4.在该文件夹下的终端运行./frpc -c ./frpc.ini

5.记得把域名服务器开放相应的端口，具体教程可以看：
https://www.iplaysoft.com/frp.html
