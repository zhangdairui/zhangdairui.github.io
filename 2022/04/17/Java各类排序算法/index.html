快速排序 Quick Sort

核心思想

最核心的目的就是在确定基准点之后，将小于基准点的数字放在它的左侧，大于的放在右侧。最后递归整个过程。每一轮排序结束的标志就是，将基准点安顿在“中间”的正确位置。

具体的操作是：

  1. 选择一个key(或者叫pivot)，当做整个数列的基准。一般是数列的起点元素。
  2. 准备一头一尾两个指针：x和y。
  3. x其实是负责从前往后，找到比key大的值，因为它不应该在左边。反之，y负责从后往前，找到本不应该在右边的，比key小的值。*这里所说的该不该在左或右，是根据key点在正中间去想象的，至于如何把key点放在所谓的“正中间”，这正是一次遍历结束后我们要做的事情。
  4. 先操作y，向前找，找到比key小的小的值。停住。
  5. 再操作x，向后找，找到比key大的值，停住。
  6. 交换x和y指针指向的数组的值
  7. 最终x和y指针相遇，证明这一圈的操作结束，此时把key安放到x或者y的位置。
  8. 递归上述步骤。注意3-5，还可以轮流洗掉（下文方法二。）

代码实现

代码实现其实可以选择两种方式：

  • 第一种方式，指针停下后交换。

    移动完y就移动x，指针都停住之后，交换x处和y处的数值。注意此方式，x的起始位置是head+1，也就是第二个元素。因为第一个元素就是key本身，比较也没有意义。最终，直至x和y相遇时，我们要讲相遇处的值，与从未动过的head交换。这也等于，是让身为key的head提前等待，等着我们把smaller的数和larger的数都挑出来，最后再把head放到中间。

  • 第二种方式，x和y互相洗掉。

    移动完y，y停住之后，将y处的值赋给x处。把x处的值洗掉。再去动x指针，x停住之后，把x处的值赋给y。注意此方法，x的起始位置就是head，因为，我们需要head处值被洗掉，空出来位置。只有head值被洗掉了没关系，因为key替我们保存着head值。直至最后，从key里把保存的值拿出来，放到x和y相遇的位置。就可以结束。

    public static void quickSort(int[] nums, int left, int right){

            //快速排序的每一轮结束标志：把key安顿好。

            int y = right;

                //找到了，赋给y指针的位置

                nums[y] = nums[x];

            nums[y] = key;

          quickSort(nums , y+1 , right);

        }

需要注意的关键点

  • 快速排序，我设定的key在head处（数组头部），是一定要先挪y的，从后面开始找。这个原因，忘了的话看dk的视频。在电脑桌面上的【重要视频】文件夹里。

    原因如下：

    方法一：如果先挪x，最终x和y交汇的点肯定是：不应该在左边的、数值很大的点。因为这就是x的使命！这样的点，再去和head交换，最大的点反而占据了最前面的位置。没有意义。

方法二：如果先挪x，会导致后面的值在第一次洗的时候就洗掉了，没有保存。先挪y，第一次洗x值时候，x的值被保存在key里，后面可以拿回来。

  • x指针的起始位置，两种方法是不同的。轮流洗掉需要先让head被key保存，空出位置。同时交换需要head不要动，等到最后去交换。

  • 每次在挪x或y指针之前，都要判断一下，此时x是不是等于y了，如果是的话，随时退出循环。因为指针相遇了这一圈就遍历结束了。不然挪过头了，遇到与key相等的值，就会出现错误。

选择排序 Selection Sort

核心思想

每一次都找到剩余未排序数组中最小的数放进去。

代码

class Solution {
//Selection sort 选择排序。
//每次都选择当前数列中最小的元素。
    public static void selectionSort(int[] nums, int curr){

        if(curr >= nums.length - 1) return;

        int min = nums[curr];
        int minCurr = curr;
        for(int i = curr+1 ; i<nums.length ; i++){
            if(nums[i] < min){
                min = nums[i];
                minCurr = i;
            }

        }

        //已经找出最小值min
        //将min和curr位的数组交换。
        nums[minCurr] = nums[curr];
        nums[curr] = min;


        curr++;
        selectionSort(nums, curr);
    }


// Driver！！！！
    public int[] sortArray(int[] nums) {

        selectionSort(nums , 0);

        return nums;

    }
}

插入排序 Insert Sort

核心思想

把每个数都插入到合适的位置。不断地和前面一个对比，如果更小的话就往前串（交换），以此类推。
时间复杂度O(n^2)

代码实现

class Solution {
//Insertion sort 插入排序。

//核心思想： 每次都将当前元素插入

    public int[] sortArray(int[] nums) {

        for(int i = 0; i< nums.length; i++){
            for(int j = i; j>0; j--){
                if(nums[j-1] > nums[j]){
                    int temp = nums[j];
                    nums[j] = nums[j-1];
                    nums[j-1] = temp;
                }else{
                    break;
                }

            }
        }


        return nums;

    }
}
