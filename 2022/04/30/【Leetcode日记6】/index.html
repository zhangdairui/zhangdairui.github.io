判断是否有环

准备两个指针，一个快（每次循环向后挪2步）一个慢（每次循环向后挪1步）。

public class Solution {
    public boolean hasCycle(ListNode head) {
        if(head == null) return false;
        ListNode slow = head;
        ListNode fast = head.next;
        while(slow != null && fast != null && slow.next != null && fast.next!=null){
            if(slow == fast) return true;
            slow = slow.next;
            fast = fast.next.next;
        }
        return false;
    }
}

寻找环的起点

第一步：判断是不是有环。第二步：寻找环的起点

关于寻找环的起点：

一个指针从head走，

一个指针从之前快慢指针相遇的点走，

它们相遇的点就是环的起点。

public class Solution {
    public ListNode detectCycle(ListNode head) {
        if(head == null || head.next == null) return null;
        //先看有没有cycle
        ListNode a = head;
        ListNode b = head;

        Boolean cycle = false;
        while(b!=null && b.next!=null){
            a = a.next;
            b = b.next.next;
            if( a == b){
                cycle = true;
                break; //有环
            }
        }

        if(!cycle) return null;

        a = head;
        while(b != null){

            if(a == b) return a;
            a = a.next;
            b = b.next;

        }

        return null;

    }
}
