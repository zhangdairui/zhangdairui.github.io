在pycharm下搭建Hadoop环境连接HDFS

  • 安装big data tool插件，步骤见官网指南
  • 安装好后重启IDE，在右侧出现了big data tools选项卡，点开，点击加号可以配置HDFS。

问题详情

在伪分布模式下，执行

python wordcount.py -r hadoop hdfs:///input/SampleTextFile_1000kb.txt -o hdfs:///output
时候报如下错误：

Error: java.net.ConnectException: Call From localhost/127.0.0.1 to localhost:8020 failed on connection exception: java.net.ConnectException: Connection refused;

解决方式

  • 首先jps检查是不是所有的东西都启动了，有的时候是因为namenode或者datanode没有启动。
  • cat /etc/hosts 看看hosts配置有没有写错的。包括自己的ip地址，localhost有没有对应好等。
  • 如果还是不好的话，将core-site.xml文件中的localhost:8020（一般网上的教程都是要配置成这个localhost的形式）改成如下，然后重启hadoop集群即可。
    <property>
   <name>fs.defaultFS</name>
   <value>hdfs://127.0.0.1:8020</value>
</property>
