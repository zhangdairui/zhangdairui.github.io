代码

如果纯递归会超时（递归太深栈溢出）。
就直接准备一个数组记录下每一次计算完fibResult的结果。

/**
 * @param {number} n
 * @return {number}
 */
var fib = function(n) {
     var fibResult = new Array(n+1); //用来记录fib每一项的结果。
     fibResult[0]=0;
     fibResult[1]=1;

     for(var i = 2;i<n+1;i++){
        fibResult[i] = (fibResult[i-1]+fibResult[i-2])%1000000007;
     }

    return fibResult[n];

};

其实更好的方法是不用数组，准备3个参数，随着指针往后挪 变成三个元素之间的问题。
以下是leetcode上的解答：

var fib = function(n) {
    let n1 = 0, n2 = 1, sum;
    for(let i = 0; i < n; i++){
        sum = (n1 + n2) % 1000000007;
        n1 = n2;
        n2 = sum;
    }
    return n1;
};
