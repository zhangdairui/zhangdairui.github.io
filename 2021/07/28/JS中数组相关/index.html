JS中数组相关
emo | 2021年07月28日 | js

创建数组

var arr = [ ]; //创建一个空数组

var arr = new Array();　//创建一个空数组

var arr = new Array([size]);　//创建一个指定长度的数组。后面可以变化。

var arr = new Array(element0, element1, ..., elementn);　//创建一个数组并赋值

数组查找

arr.indexOf("apple");//返回apple元素所在的索引

数组插入

arr.push("apple");//向末尾添加apple元素（返回新的长度）此方法改变数组的长度

arr.unshift("apple");//向开头添加apple元素（返回新的长度）此方法改变数组的长度

数组删除

arr.pop();//删掉数组末尾元素（返回该元素）此方法改变数组的长度

arr.shift();//删掉数组第一个元素（返回该元素）此方法改变数组的长度

数组排序

//-------字母排序
arr.sort();//升序排序。

//-------数字排序
var points = [40,100,1,5,25,10];
points.sort(function(a,b){return a-b}); //升序排列
points.sort(function(a,b){return b-a}); //降序排列

//-------倒序
arr.reverse(); //返回数组地址

数组截取，合并，复制

//-------截取
arr.slice(start, end);
/*返回一个新的数组，包含从 start (包含) 到 end（不包含）的 arr中的元素。
start参数必需。规定从何处开始选取。end非必须，如果不指定默认到最后。
如果参数是负数，那么它表示倒着查。也就是说，-1 指最后一个元素，-2 指倒数第二个元素…。
*/

//-------合并
arr.concat(arrayX,arrayX,......,arrayX);//arrayX参数必须，可以是单个元素的值，也可以是数组。在arr尾部添加，返回新数组。

//-------复制
arr.slice(0); 

arr.concat();

Array对象可用方法一览

方法 功能
entries() 返回数组的可迭代对象。
concat() 连接两个或更多的数组，并返回结果。
copyWithin() 从数组的指定位置拷贝元素到数组的另一个指定位置中。
every() 检测数值元素的每个元素是否都符合条件。
fill() 使用一个固定值来填充数组。
filter() 检测数值元素，并返回符合条件所有元素的数组。
find() 返回符合传入测试（函数）条件的数组元素。
findIndex() 返回符合传入测试（函数）条件的数组元素索引。
forEach() 数组每个元素都执行一次回调函数。
from() 通过给定的对象中创建一个数组。
includes() 判断一个数组是否包含一个指定的值。
indexOf() 搜索数组中的元素，并返回它所在的位置。
isArray() 判断对象是否为数组。
join() 把数组的所有元素放入一个字符串。
keys() 返回数组的可迭代对象，包含原始数组的键(key)。
lastIndexOf() 搜索数组中的元素，并返回它最后出现的位置。
map() 通过指定函数处理数组的每个元素，并返回处理后的数组。
pop() 删除数组的最后一个元素并返回删除的元素。
push() 向数组的末尾添加一个或更多元素，并返回新的长度。
reduce() 将数组元素计算为一个值（从左到右）。
reduceRight() 将数组元素计算为一个值（从右到左）。
reverse() 反转数组的元素顺序。
shift() 删除并返回数组的第一个元素。
slice() 选取数组的一部分，并返回一个新数组。
some() 检测数组元素中是否有元素符合指定条件。
sort() 对数组的元素进行排序。
splice() 从数组中添加或删除元素。
toString() 把数组转换为字符串，并返回结果。
unshift() 向数组的开头添加一个或更多元素，并返回新的长度。
valueOf() 返回数组对象的原始值。
