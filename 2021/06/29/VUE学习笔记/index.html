VUE学习笔记
emo | 2021年06月29日 | 前端

Vue使开发过程中无需直接操作dom元素，并实现组件化开发。此篇是初学Vue过程中的一些笔记。供自己加深印象。查缺补漏。

基础知识

首先：

  1. 官方文档：
    VUE中文文档
  1. 引入cdn：
    <!-- 开发环境版本，包含了有帮助的命令行警告 -->
<script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
<!-- 生产环境版本，优化了尺寸和速度 -->
<script src="https://cdn.jsdelivr.net/npm/vue@2"></script>

常用基本Vue指令：

v-text：更改元素内部文字（除非使用表达式，不然默认全部替换）

v-html：更改元素的html代码

v-on：可写成@，为元素绑定事件。如:

@click = "functionName"
@keyup.enter = "functionName(parameter)" //指定回车键触发事件

v-show：控制元素是否显示

v-if：控制元素是否存在。（较耗性能，因此在仅需要控制显示的情况下，v-show更好。）

v-bind：操作元素的属性。简写成冒号，如：:scr=”imgAddress”。

v-for：遍历准备好的数组数据在网页中生成ul列表结构。在li标签内部用双花括号写好要用的元素。

如：

<li v-for="item in arr">{{item.name}}</li>

或者直接可以使用index：

<li v-for="(item, index) in arr"> {{ index }} </li>

v-model：获取/设置input表单元素的值。一旦用户更新了input中的值，在Vue实例中也会同步更新。

Demo组件

点击箭头图片轮播（v-bind + v-show + v-on）

HTML代码：

<div id="mask"> 
      <div class="center">
        <h2 class="title"><img src="./images/logo.png" alt=""> 深圳创维校区环境</h2>
        <img :src="imgList[index]" alt="" />
        <a
          href="javascript:void(0)"
          @click="prev"
          class="left"
          v-show="index>0"
        >
 <!--上方a标签是一个向左指的箭头，@click绑定点击事件，
v-show指定一个表达式，当index=0的时候（也就意味着轮播到了第一张图片）
需要将该箭头隐藏。-->

          <img src="./images/prev.png" alt="" />
        </a>
        <a
          href="javascript:void(0)"
          @click="next"
          class="right"
          v-show="index<imgList.length-1"
        >
          <img src="./images/next.png" alt="" />
        </a>
      </div>
    </div>

Vue部分代码：

<script>
      const app = new Vue({
        el: "#mask",
        data: {
          imgList: [
            "./images/00.jpg",
            "./images/01.jpg",
            "./images/02.jpg",
            "./images/03.jpg",
            "./images/04.jpg",
            "./images/05.jpg",
            "./images/06.jpg",
            "./images/07.jpg",
            "./images/08.jpg",
            "./images/09.jpg",
            "./images/10.jpg",
          ], //数组保存需要轮播的图片地址
          index: 0 
        },
        methods: {
          // 上一张
          prev() {
            this.index--; //this代表当前vue实例
          },
          // 下一张
          next() {
            this.index++;
          }
        }
      });
    </script>
